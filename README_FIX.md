# Быстрое решение проблемы с пустыми файлами DAO

## Проблема
IntelliJ IDEA не видит содержимое файлов DAO интерфейсов.

## Решение (выполните по порядку):

### 1. Очистка кэша IDE (ОБЯЗАТЕЛЬНО!)
```
File → Invalidate Caches / Restart → Invalidate and Restart
```

### 2. Пересборка проекта
```
Build → Rebuild Project
```

### 3. Если не помогло - пересоздание конфигурации
1. Закройте IntelliJ IDEA
2. Удалите папку `.idea` в корне проекта
3. Откройте проект заново
4. IntelliJ IDEA пересоздаст конфигурацию

### 4. Проверка файлов
Убедитесь, что файлы существуют в:
```
src/main/java/com/example/insurancecompany/dao/
```

Все файлы DAO должны содержать код интерфейсов.

## Важно!
Файлы DAO **существуют и содержат правильный код**. Проблема только в индексации IDE.

После выполнения шагов 1-2 ошибки должны исчезнуть.
